Load times

DOM Loaded - The initial HTML document has been completely loaded and parsed, without waiting for stylesheets, images, and subframes to finish loading.

DOM Complete - The time represents the end of the browser processing a document. The browser has received the document, processed it and has done the same for the page resources like images, CSS and JavaScript.

  1. To the End user, this is the point where the browser completes loading (progress bar or spinner stops spinning)
  2. To a developer, this marks the beginning of the time to add additional application logic (i.e. Javascript)
